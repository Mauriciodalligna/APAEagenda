# Criar PRD (APAE Agenda)

Segue **integralmente** o fluxo em `.cursor/skills/criar-prd/SKILL.md` (esclarecer → planejar → redigir → salvar).

- Não gere o PRD antes da fase de perguntas de clarificação.
- Template: `templates/prd-template.md`. Saída: `tasks/prd-[nome-funcionalidade]/prd.md`.
- Carregue referências da skill (`references/`) conforme cada fase.
