# Executar task — PRD de produto (APAE Agenda)

Segue **integralmente** o fluxo em `.cursor/skills/executar-task/SKILL.md`.

- Pasta: `tasks/prd-[nome-funcionalidade]/` (se não disser qual, use contexto ou pergunte).
- Próxima tarefa: `tasks.md` e `[num]_task.md` — primeira pendente.
- Fluxo: configuração pré-tarefa → análise → resumo → plano → implementação. Use **npm** e scripts do `package.json`.

Para **refatoração** (`tasks/prd-refatoracao-*`), use o comando **ref-executar-task**.
